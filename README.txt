TOXICIFY - World of Warcraft Addon
====================================

OVERZICHT
---------
Toxicify is een krachtige World of Warcraft addon die je gameplay ervaring verbetert met geavanceerde speler tracking en monitoring functionaliteiten.

HOOFDFUNCTIES
-------------
• Geavanceerde speler tracking en monitoring
• Aanpasbare UI met modern design
• Group finder integratie
• Real-time speler statistieken
• Eenvoudige configuratie opties
• Minimap integratie
• Player database met toxiciteit tracking

INSTALLATIE
-----------
1. Download de Toxicify installer
2. Voer de installer uit
3. Selecteer je WoW AddOns directory
4. De installer maakt automatisch een 'Toxicify' folder aan
5. Start WoW en activeer de addon in het AddOns menu

GEBRUIK
-------
• Rechts-klik op spelers om ze toe te voegen aan je database
• Gebruik de minimap knop voor snelle toegang
• Configureer instellingen via het options menu
• Bekijk speler statistieken in real-time
• Gebruik de group finder integratie voor betere groepen

COMMANDO'S
----------
/toxic - Hoofdcommando voor alle functies

Basis commando's:
/toxic add <naam-realm>        - Markeer speler als Toxic
/toxic addpumper <naam-realm>  - Markeer speler als Pumper  
/toxic del <naam-realm>        - Verwijder speler uit lijst
/toxic list                    - Toon huidige lijst
/toxic ui                      - Toggle Toxicify venster
/toxic settings                - Open addon instellingen
/toxic config                  - Open addon instellingen (alias)

Geavanceerde commando's:
/toxic export                  - Exporteer lijst (string)
/toxic import <string>         - Importeer lijst van string
/toxic debug                   - Toggle debug modus
/toxic partywarning            - Toggle party waarschuwing
/toxic luaerrors               - Toggle Lua errors (vereist debug)
/toxic contextmenu             - Activeer context menu marking
/toxic testwarning             - Toon test waarschuwing popup
